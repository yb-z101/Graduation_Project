# 用户操作手册框架

## 一、系统简介

### 1.1 系统概述
对话式智能数据分析系统是一款基于大模型自然语言理解能力与工作流流程编排的B/S架构Web系统。用户可以通过自然语言交互方式完成数据分析，无需专业编程技能。

### 1.2 主要功能
- 多源数据上传（Excel、CSV、SQL文件）
- 自动数据清洗
- 自然语言数据分析
- 多轮对话记忆
- 可视化图表展示（饼图、折线图、柱状图等）
- 会话管理
- 数据库连接与查询

### 1.3 技术架构
- 前端：Vue 3 + Element Plus + ECharts
- 后端：FastAPI + Python + Pandas + LangGraph
- 数据库：MySQL

## 二、快速开始

### 2.1 系统要求
- 浏览器：Chrome 90+、Firefox 88+、Edge 90+、Safari 14+
- 网络：稳定的互联网连接（用于调用大模型API）

### 2.2 首次使用
1. 打开浏览器，访问系统地址
2. 进入主界面
3. 上传数据文件或连接数据库
4. 开始自然语言对话分析

## 三、功能详细说明

### 3.1 文件上传

#### 3.1.1 支持的文件格式
- **Excel文件**：.xlsx、.xls格式
- **CSV文件**：.csv格式
- **SQL文件**：.sql格式

#### 3.1.2 上传步骤
1. 点击页面底部的"上传文件"按钮
2. 选择要上传的文件
3. 等待文件上传和解析完成
4. 系统自动进行数据清洗
5. 查看文件预览信息

#### 3.1.3 注意事项
- 文件大小限制：10MB以内
- 建议先预览数据再进行分析
- SQL文件会自动执行并创建临时表

### 3.2 数据库连接

#### 3.2.1 连接数据库
1. 点击"数据库连接"按钮
2. 填写数据库连接信息：
   - 主机地址
   - 端口
   - 数据库名
   - 用户名
   - 密码
3. 点击"连接"按钮
4. 查看数据库表结构

#### 3.2.2 Chat to SQL功能
1. 连接数据库后，直接用自然语言提问
2. 系统自动生成SQL语句
3. 执行查询并展示结果
4. 可选择生成可视化图表

### 3.3 自然语言分析

#### 3.3.1 基本查询
- 聚合计算："计算总分的平均值"、"求和"
- 排序："按成绩降序排列"、"从高到低排序"
- 筛选："筛选成绩大于80分的学生"、"只看一班的数据"
- 分组统计："按班级分组计算平均分"

#### 3.3.2 多轮对话
系统支持多轮对话记忆，上下文保持有效轮数至少3轮：
- 第1轮："查看所有学生数据"
- 第2轮："只看一班的"
- 第3轮："计算他们的平均分"

#### 3.3.3 提示技巧
- 明确指出要分析的列名
- 说明计算方式（求和、平均、最大值等）
- 指定筛选条件
- 如需要图表，明确说明图表类型

### 3.4 可视化图表

#### 3.4.1 支持的图表类型
- **饼图**：适用于占比分析
- **折线图**：适用于趋势分析
- **柱状图**：适用于对比分析
- **散点图**：适用于相关性分析
- **雷达图**：适用于多维度分析
- **表格**：适用于详细数据展示

#### 3.4.2 生成图表
在对话中明确要求图表类型：
- "用饼图展示各班人数占比"
- "生成折线图展示成绩趋势"
- "用柱状图对比各班平均分"

### 3.5 会话管理

#### 3.5.1 创建新会话
点击左侧侧边栏的"新会话"按钮，即可创建新的分析会话。

#### 3.5.2 切换会话
在左侧会话列表中点击任意会话，即可切换到该会话。

#### 3.5.3 删除会话
1. 鼠标悬停在会话上
2. 点击删除图标
3. 确认删除操作

#### 3.5.4 会话持久化
所有会话数据会自动保存，刷新页面后可恢复。

### 3.6 模型切换

系统支持多个大模型，可在顶部栏切换：
- 阿里云 Qwen Turbo
- DeepSeek R1
- 火山引擎 Doubao
- 星火大模型

## 四、常见问题解答

### 4.1 文件上传相关

**Q: 上传文件后没有反应？**
A: 请检查文件格式是否正确，文件大小是否超过10MB限制。

**Q: 数据解析错误怎么办？**
A: 请确保文件编码为UTF-8，CSV文件使用逗号分隔。

### 4.2 数据分析相关

**Q: 分析结果不准确？**
A: 尝试更明确地描述您的需求，指明具体的列名和计算方式。

**Q: 多轮对话上下文丢失？**
A: 请确保在3轮对话内，系统会自动保存最近的对话历史。

### 4.3 图表生成相关

**Q: 为什么没有生成图表？**
A: 请明确说明需要的图表类型，确保数据包含数值列。

**Q: 图表可以导出吗？**
A: 目前暂不支持直接导出，可使用浏览器截图功能。

### 4.4 数据库连接相关

**Q: 连接数据库失败？**
A: 请检查连接信息是否正确，数据库服务是否启动，网络是否通畅。

**Q: SQL文件执行失败？**
A: 请确保SQL语法正确，不包含DROP、DELETE等危险操作。
